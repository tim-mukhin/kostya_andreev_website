# Академия Кости Андреева — сайт школы BMX

Статический сайт-визитка BMX-школы Кости Андреева (Москва, Rampstroy).
Чистый HTML + CSS + немного JS, без сборки. Работает на десктопе и мобилке.

## Страницы
- `index.html` — главная (о школе, программы, тренер, галерея)
- `training.html` — тренировки, расписание, цены, форма записи
- `shop.html` — мерч (предзаказ)
- `about.html` — о школе и тренере

## Ассеты
- `assets/logo/` — логотип: полный/знак, чёрный/белый, фавиконки
- `assets/img/` — фото (мокапы бренда)
- `styles.css`, `app.js` — общие стили и скрипты

## Деплой (GitHub Pages)
Settings → Pages → Source: **Deploy from a branch** → `main` / `root`.
Сайт откроется на `https://<user>.github.io/kostya_andreev_website/`.
Все пути относительные — работает из подпапки проекта.

## Заглушки (заменить перед запуском)
- телефон, email, ссылки Telegram/VK/YouTube в футере
- цены в `training.html` и `shop.html`
- форма записи сейчас без бэкенда (показывает «спасибо»); подключить к Telegram-боту или Formspree
